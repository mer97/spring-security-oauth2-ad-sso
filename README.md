"# spring-security-oauth2-ad-sso" 

基于Spring Boot2、Active Directory域认证、Spring Security OAuth2 实现的单点登录（SSO）系统

demo包含了一个SSO服务端（sso-server）和两个客户端（client-user、client-order）

我的环境是IDEA、Maven、jdk1.8
